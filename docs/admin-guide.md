# Admin Guide

## Overview

AttendX is a face recognition attendance platform that integrates with Hikvision terminals for automated attendance tracking.

## Features

- Student management with face enrollment
- Hikvision terminal integration (ISAPI protocol)
- Real-time attendance tracking
- Telegram bot notifications for parents
- Comprehensive attendance reports
- Role-based access control

## User Roles

- **Super Admin** — Full system access
- **Admin** — Manage students, devices, and view reports
- **Viewer** — View attendance records and reports only
